# Presidential Rice for George Washingtons Birthday week

* this is a small repo to create a rice of dotfiles

## Toolchain to rice

* hyprland
* neovim
* fish 
* waybar 
* hyprlock 
* hypridle 
* hyprlauncher 
* hyprpaper
* fonts (noto is usually my default but maybe we can go for something more classy?)
  * my fonts: noto-fonts noto-fonts-cjk noto-fonts-emoji noto-fonts-extra


## Essential utilities 

* swaync 
* pipewire and wireplumber
* hyprpolkitagent 
* qt5-wayland and qt6-wayland.
* thunar (GUI) TUI (BASH)
